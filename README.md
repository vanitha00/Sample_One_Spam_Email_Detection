# Spam Email Detection

A simple Machine Learning project that classifies emails as spam or not spam using a classification algorithm.

## Features
- Email text preprocessing
- Spam email classification
- Model training and prediction
  
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
